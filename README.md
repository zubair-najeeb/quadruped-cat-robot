# Quadruped Cat Robot

A quadruped robotic system built using Arduino Nano and SG90 servo motors to simulate realistic walking motion.

## Overview

This project demonstrates the design of a four-legged robotic platform where each leg uses two servo motors (shoulder and knee joint). The robot mimics walking motion through coordinated servo control.

## Hardware Components

Arduino Nano
8x SG90 Servo Motors
Funduino Nano Shield
External Power Supply
Robot Frame
sensor tile integration
## Features

Four-leg walking mechanism
Servo coordination for realistic movement
Embedded microcontroller control
Expandable for AI navigation and sensors

## Future Improvements

Obstacle detection
Camera based navigation
AI powered movement control
